# Portfolio (GitHub Pages) – TeXt Theme

Dieses Repo ist ein **fertiges Grundgerüst** für ein Bewerbungs‑Portfolio (Tierpflege / pädagogischer Bauernhof) auf **GitHub Pages** – mit dem Jekyll‑Theme **TeXt**.

## Schnellstart (in 5 Minuten)
1. Repo auf GitHub anlegen:
   - **empfohlen**: `USERNAME.github.io` (Public)
2. Diese Dateien hochladen (oder `git push`).
3. GitHub → **Settings → Pages**
   - Source: *Deploy from a branch*
   - Branch: `main` / root
4. Nach 1–3 Minuten ist die Seite online:
   - `https://USERNAME.github.io/` (oder Repo‑Pages URL)

## Inhalte anpassen
- `_config.yml`:
  - `title`, `description`, `author.name`, Navigation
- Seiten:
  - `ueber-mich.md`, `erfahrung.md`, `tiere-und-aufgaben.md`, `paedagogik.md`, `galerie.md`, `videos.md`, `kontakt.md`
- Bilder:
  - in `assets/img/` ablegen
  - in Markdown per `![Alt-Text](/assets/img/datei.jpg)` einfügen

## Videos (Empfehlung)
Bitte **nicht** große Videodateien in dieses Repo hochladen.
Besser:
- YouTube „nicht gelistet“ oder
- Nextcloud/Google Drive/OneDrive
und hier nur **Links + QR‑Code‑Bilder** verwenden.
